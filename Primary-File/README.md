# Codeageing

Simple scaffold for the Codeageing software development agency website.

Theme: brand dark `#344C36`, accent `#FAAD1B`.

Files:
- `index.html` — main HTML page
- `style.css` — styles using the brand colors
- `app.js` — minimal JS for contact form

How to run locally:
1. Open `index.html` in a browser.
