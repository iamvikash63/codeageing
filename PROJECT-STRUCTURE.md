# Codeageing Structure

## Current Main Files
- `index.html`: Home page
- `about.html`: About page
- `courses.html`: Courses landing page
- `style.css`: Home page styles
- `about.css`: About page styles
- `courses.css`: Courses page styles
- `app.js`: Home page interactions
- `courses.js`: Courses page interactions
- `Primary-File/Images`: Project images and brand assets
- `Primary-File/drop-down`: Dropdown and service assets

## Safe Approach
- Keep the current structure for now so links do not break.
- Move files in one step later only after updating all HTML paths.
- Keep each page paired with one CSS file and one JS file.
- Store all future images inside one asset folder only.
